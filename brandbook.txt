SPACING SYSTEM (MA)

Horizontal padding
24px минимум

Vertical spacing between major blocks
24-40px

Button height
48-52px

Border radius
12-16px

Rules:

Один главный CTA на экран

Secondary CTA только текстовой ссылкой

Не использовать разделители если можно использовать пустоту

============================================================

COMPONENT SYSTEM

5.1 Buttons

Primary Button:

Background: Accent Primary

Text: Background Primary

No shadow

Radius: 12-16px

Height: 48-52px

Secondary Button:

Text-only or subtle outline

Цвет: Primary Text или Accent

No gradients
No heavy shadows
No glassmorphism

5.2 Breathing Timer

Core element - минималистичный круг.

Specifications:

Stroke width: 2-3px

Color: Accent Primary

No glow

No gradients

No 3D

Animation:

Ease-in-out

Slightly slower than default system animation (~10-15%)

Expansion on inhale

Hold - static

Contraction on exhale

Circle must feel organic, not mechanical.

5.3 Phase Labels

Text examples:

Inhale

Hold

Exhale

Begin

Complete

No emojis
No exclamation marks
No motivational copy

============================================================

MOTION SPECIFICATION

General rules:

10-15% slower than default iOS timing

Prefer fade over slide

No bounce

No aggressive spring

Transitions:

Screen change - subtle fade
State change - opacity + scale max 1.02

Animation should feel like breathing, not UI feedback.

============================================================

SOUND DESIGN

Soft wooden gong

Subtle tick during hold phase

No ambient music by default

No artificial meditation sounds

Sound should be optional.

============================================================

SCREEN STRUCTURE

8.1 Home Screen

Elements:

Title

Selected breathing pattern

Primary CTA (Start)

Minimal secondary link

No banners
No ads
No carousels

8.2 Session Screen

Elements:

Breathing circle (centered)

Phase label

Timer

Minimal pause control

No statistics visible during session.

8.3 Statistics Screen

Minimal data only:

Total minutes

Current streak

Sessions count

No gamification
No badges
No social comparison

============================================================

PREMIUM PRINCIPLES

Premium feeling achieved through:

Generous spacing

Warm neutral tones

Predictable layout

Consistent typography

Calm motion

Premium does NOT mean:

Complex UI

Bright colors

Heavy visuals

Decorative Japanese elements

No sakura
No bamboo patterns
No calligraphy textures
