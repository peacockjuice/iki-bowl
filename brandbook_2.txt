1. СИСТЕМНЫЕ ОГРАНИЧЕНИЯ

Дизайн должен учитывать:

Нет React/Vue state animation

Нет CSS framework

Нет сложных компонентных библиотек

Статический деплой

Минимальный JS

Следствие:

Минимум сложных визуальных эффектов

Без glassmorphism

Без градиентов

Без heavy shadow stacks

Без pure white #FFFFFF

Без pure black #000000

Стиль должен быть реализуем чистым CSS.

2. DESIGN TOKENS (ОБЯЗАТЕЛЬНО В styles.css)
2.1 Цвета
:root {

/* Light */
--bg: #F4F2EC;
--surface: #EAE6DD;
--text: #1F1F1C;
--text-muted: #6B6A63;
--accent: #7C8A7A;
--accent-soft: #8F9C8F;

/* Dark */
--bg-dark: #1E1D1A;
--surface-dark: #2A2925;
--text-dark: #E6E2D8;
--text-muted-dark: rgba(230,226,216,0.72);
--stroke-dark: rgba(255,255,255,0.12);

}

Контраст минимум WCAG AA.

2.2 Радиусы
--radius-card: 20px;
--radius-control: 16px;
--radius-pill: 999px;

Радиус не должен быть слишком округлым.
Не iOS bubbly.

2.3 Отступы
--space-1: 8px;
--space-2: 12px;
--space-3: 16px;
--space-4: 24px;
--space-5: 32px;
--space-6: 40px;

Главный ритм:

24px — основной

32px — крупные секции
